# Grading Rubrics

**Comprehensive evaluation criteria for all assessment types**

## ðŸ“š Overview

This directory contains detailed grading rubrics for various types of assessments. Rubrics provide clear, transparent criteria for evaluation and help ensure consistent, fair grading.

## ðŸŽ¯ Rubric Philosophy

Our rubrics are designed to:

1. **Provide Clarity**
   - Clear expectations before submission
   - Transparent grading criteria
   - Specific performance levels

2. **Ensure Fairness**
   - Objective evaluation criteria
   - Consistent application
   - Reduce subjective bias

3. **Support Learning**
   - Identify strengths and weaknesses
   - Guide improvement
   - Align with learning objectives

4. **Facilitate Communication**
   - Common language for feedback
   - Clear performance standards
   - Actionable guidance

## ðŸ“Š Rubric Structure

Each rubric follows this format:

```markdown
# [Assessment Type] Rubric

## Criterion 1: [Name] (X points)

### Excellent (90-100% of points)
- Specific indicators of excellent performance
- Clear examples

### Good (80-89% of points)
- Specific indicators of good performance
- What distinguishes from excellent

### Satisfactory (70-79% of points)
- Minimum acceptable performance
- Key requirements met

### Needs Improvement (60-69% of points)
- Significant gaps or errors
- Does not meet minimum requirements

### Unacceptable (0-59% of points)
- Major deficiencies
- Does not demonstrate basic understanding
```

## ðŸŽ“ Using Rubrics

### For Educators

**Before Assignment:**
1. Share rubric with students
2. Discuss expectations and criteria
3. Provide examples at different levels
4. Answer questions about rubric

**During Grading:**
1. Evaluate each criterion independently
2. Record scores and comments
3. Be consistent across students
4. Provide specific feedback

**After Grading:**
1. Return with completed rubric
2. Discuss common issues
3. Highlight excellent examples
4. Offer improvement guidance

**Customization:**
- Adjust weights to match priorities
- Add/remove criteria as needed
- Modify performance levels
- Include course-specific elements

### For Students

**Before Starting:**
1. Read rubric carefully
2. Understand all criteria
3. Note point allocations
4. Identify what "excellent" looks like

**During Work:**
1. Use rubric as checklist
2. Self-assess as you go
3. Ensure all criteria addressed
4. Aim for higher performance levels

**After Submission:**
1. Review graded rubric carefully
2. Understand score on each criterion
3. Note patterns in feedback
4. Apply lessons to future work

### Grade Scale

| Percentage | Letter | Performance |
|-----------|--------|-------------|
| 93-100 | A | Exceptional |
| 90-92 | A- | Excellent |
| 87-89 | B+ | Very Good |
| 83-86 | B | Good |
| 80-82 | B- | Above Average |

| 73-76 | C | Satisfactory |
| 70-72 | C- | Below Average |

### Criterion-Level Assessment

**Excellent (90-100%):**
- Exceeds all requirements
- Demonstrates deep understanding
- High quality throughout
- Minimal to no errors

**Good (80-89%):**
- Meets all requirements
- Demonstrates solid understanding
- Good quality overall
- Minor errors or omissions

**Satisfactory (70-79%):**
- Meets most requirements
- Demonstrates basic understanding
- Acceptable quality
- Some significant gaps

**Needs Improvement (60-69%):**
- Meets some requirements
- Limited understanding
- Below acceptable quality
- Major gaps or errors

**Unacceptable (0-59%):**
- Does not meet requirements
- Insufficient understanding
- Poor quality
- Pervasive errors


### Best Practices

1. **Clear Criteria**
   - Specific and measurable
   - Observable behaviors
   - Avoid vague language

2. **Appropriate Granularity**
   - Not too detailed (overwhelming)
   - Not too broad (unclear)
   - Actionable descriptions

3. **Balanced Weighting**
   - Reflect importance
   - Align with learning objectives
   - Total to 100 points

4. **Objective Language**
   - Factual descriptions
   - Avoid subjective judgments
   - Clear distinctions between levels

## ðŸ’¡ Grading Tips

### Consistency

- Grade same criterion across all submissions
- Use examples to calibrate
- Take breaks to maintain focus
- Review grading periodically

### Feedback

- Be specific and constructive
- Point to resources
- Highlight strengths
- Suggest improvements

### Common Issues

**Watch For:**
- Grade inflation/deflation
- Halo effect (one aspect affects all)
- Recency bias (later submissions graded differently)
- Leniency/severity bias

**Solutions:**
- Blind grading when possible
- Multiple graders for calibration
- Regular calibration discussions
- Rubric adherence
