## [1.0.0] - 2025-01-05

### Added
- Initial release of Academic Presentations repository
- **15+ comprehensive presentations** covering:
  - Deep Learning Fundamentals
  - Reinforcement Learning
  - Explainable AI & Model Interpretability
  - Time Series Analysis
  - Optimization for Data Science
  - MCMC Methods
  - Bayesian Machine Learning
  - Causal Inference
  - Statistical Learning Theory
  - Feature Engineering
  - A/B Testing & Experimentation
- **ESMAD Beamer Theme** with professional styling
- **27,000+ lines of production code** (Python & R):
  - MCMC implementations (Metropolis-Hastings, HMC, NUTS)
  - Causal inference methods (IV, RDD, DiD)
  - Feature engineering pipelines
- **140+ curated references** with DOIs in BibTeX format
- **Comprehensive exercise sets**:
  - MCMC exercises (8 problems, 6 pages)
  - Causal inference exercises (11 problems, 8 pages)
- **GitHub Actions automation**:
  - Automatic LaTeX compilation
  - Link checking
  - PDF preview generation
- **Documentation**:
  - Main README with learning objectives
  - CONTRIBUTING guidelines
  - Theme documentation (STYLE_GUIDE.md)
  - Code documentation

### Infrastructure
- GitHub Actions workflows for CI/CD
- Dependabot configuration
- Markdown link checker
- Automated PDF compilation pipeline

## Migration Guides

### Migrating to 1.0.0

**From 0.9.0 to 1.0.0:**

1. **Updated file structure:**
   - All presentation folders now include individual README files
   - New `assessments/` directory added
   - Updated bibliography organization

2. **Theme changes:**
   - Theme now in `theme/` directory
   - Updated macros and environments
   - See `theme/STYLE_GUIDE.md` for details

3. **Code updates:**
   - All code examples follow updated style guide
   - Python code formatted with Black
   - R code formatted with styler

**Action items:**
1. Update theme imports in custom presentations
2. Review new bibliography structure
3. Check code examples for updated style
