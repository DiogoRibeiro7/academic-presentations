# Domain 00: Programming Fundamentals

## Overview

This domain contains foundational programming materials for students who need to learn or review programming concepts before diving into data science topics.

## Topics

### R Programming
**Directory**: `r-programming/`

Comprehensive introduction to R programming for data science:
- R basics and data structures
- Data manipulation with tidyverse
- Visualization with ggplot2
- Statistical computing
- R Markdown for reproducible research

**Prerequisites**: None (beginner-friendly)
**Duration**: 2-3 weeks

## Learning Objectives

By completing this domain, students will be able to:
- Write functional R programs for data analysis
- Manipulate and transform datasets
- Create effective data visualizations
- Apply R for statistical computing
- Generate reproducible reports

## Course Integration

This domain is recommended as a prerequisite for:
- 01-Foundations (all topics)
- 03-Bayesian Methods
- 04-Causal Inference

## Materials

Each topic contains:
- `presentation/` - LaTeX Beamer slides
- `code/` - Example R scripts and notebooks
- `exercises/` - Practice problems and solutions
