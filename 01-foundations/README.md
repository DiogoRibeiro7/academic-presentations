# Domain 01: Foundations

## Overview

Core machine learning and statistical methods that form the foundation of modern data science practice.

## Topics

### 1. Statistical Modeling
**Directory**: `statistical-modeling/`

Fundamental statistical learning theory and methods:
- Supervised learning concepts
- Linear and logistic regression
- Regularization (Ridge, Lasso, Elastic Net)
- Model selection and validation
- Tree-based methods
- Ensemble methods (Random Forests, Boosting)

**Prerequisites**: Statistics, linear algebra
**Duration**: 4-5 weeks

---

### 2. Feature Engineering
**Directory**: `feature-engineering/`

Practical techniques for preparing data and creating features:
- Missing value imputation strategies
- Categorical encoding methods
- Feature scaling and normalization
- Polynomial features and interactions
- Feature selection techniques
- Pipeline construction

**Prerequisites**: Python/R basics, pandas/tidyverse
**Duration**: 2-3 weeks

---

### 3. Principal Component Analysis (PCA)
**Directory**: `pca/`

Dimensionality reduction and unsupervised learning:
- Mathematical foundations of PCA
- Eigenvalue decomposition
- Variance explained and scree plots
- Applications in visualization and preprocessing
- Extensions (Kernel PCA, Sparse PCA)

**Prerequisites**: Linear algebra
**Duration**: 1-2 weeks

---

### 4. Optimization
**Directory**: `optimization/`

Optimization methods for data science and machine learning:
- Convex optimization fundamentals
- Gradient descent variants (SGD, mini-batch)
- Momentum and adaptive methods (Adam, AdamW)
- Constrained optimization
- Evolutionary algorithms
- Bayesian optimization

**Prerequisites**: Calculus, linear algebra
**Duration**: 3-4 weeks

## Learning Path

**Recommended Order**:
1. Statistical Modeling (foundation)
2. Feature Engineering (practical skills)
3. PCA (dimensionality reduction)
4. Optimization (advanced methods)
